# Dynamic Student Grouping System

## 📌 Overview

The **Dynamic Student Grouping System** is a web application that dynamically forms student groups based on academic performance, interests, and other relevant factors. It ensures balanced and efficient team composition for academic projects, activities, or competitions.

## 🚀 Features

- 📂 **CSV-Based Data Upload**: Easily upload student data via a CSV file.
- 📊 **Performance-Based Grouping**: Ensures a mix of high, medium, and low performers in each group.
- 🎯 **Interest-Based Grouping**: Groups students based on shared interests.
- ✉️ **Automated Email Notifications**:
  - 📢 First-time updates are sent to all students.
  - 🔄 Subsequent updates are sent only to students whose sections change.

## 🛠 Tech Stack

- 🖥 **Frontend**: React.js
- ⚙️ **Backend**: Django (Python)

## 📖 Usage

1. **Upload CSV**: Admin uploads student data via the web interface.
2. **Group Formation**: The system dynamically creates balanced student groups.
3. **Email Notification**: Students receive emails about their assigned groups.
4. **Updates**: If a student’s section changes, only the affected students receive emails.
